moustacheminer server services for discord is the next generation in logical modular bots made in discord.js

Features

  • Integration with DECtalk
  • Modular file structure makes MSS-Discord easy to update
  • In built instruction manual with @MSS man [command]
  • FFMPEG to support arbitary media files

Documentation

This list of commands is non-exaustive, not fully in depth and manually updated. Please refer to the @MSS man command for the full list of commands currently available to MSS-Discord
  • 8-Ball

    8 [input]

    Send a normal 8 ball reply

  • Control

    control

    Send a reactable control panel

  • Cookie

    cookie

    Send a random fortune cookie

  • cowsay

    cowsay [input]

    Put text into a cow's speech bubble

  • Dictate

    dictate [input]

    Dictate using DECtalk

    dictate [id]

    Play a specific pre-generated DECtalk sound

  • Help

    help

    Get useful links to MSS services

  • Info

    info

    Get useful info about the bot

  • Kahoot!

    kahoot [uuid]

    Play this specific Kahoot

    kahoot https://create.kahoot.it/#quiz/[uuid]

    Play this specific Kahoot

    stop

    Stop the Kahoot game

  • Languages and Locale

    lang

    Get all locale options

    lang [language]

    Select and set your language

  • List

    list

    List all the current media on the playlist

  • Manual

    man

    List all available commands

    man [command]

    Get information about the specific command

  • Ping

    ping

    Check the delay between the bot and Discord's API servers

  • /r/Place

    place [reddit username]

    Get /r/place info for the specified username

    Get /r/place info for the specified username

  • Proverb

    proverb

    Send a random proverb

  • Roll a Dice

    roll

    Roll a single 6 sided dice.

    roll [n]

    Roll a 6 sided die [n] times.

    roll d[n]

    Roll a [n] sided dice.

    roll [n] [y]

    Roll a [y] sided die [n] times.

  • Skip

    skip

    Skip the currently playing song

  • Stop

    stop

    Stop the currently playing song

  • Play arbitary URL

    url

    Play an arbitary media URL

  • XD

    xd [message id]

    React with emojis to a specific message

  • YouTube

    yt [search]

    Search and add YouTube videos to the playlist
